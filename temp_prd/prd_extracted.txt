2. Creative Concept
2.1 The Vision
The portfolio opens like an F1 race: pitch-black screen, then an Oracle Red Bull RB19 sitting on a night circuit — from behind, engine idling. As the user scrolls, the car accelerates forward down the illuminated track. The portfolio content rises over this cinematic background layer. Every section transition is timed to the car's journey.
2.2 Cinematic References
F1 (2025 film) — cold open title sequences, dramatic track lighting
Singapore Grand Prix night race — wet asphalt, floodlights, contrast
Dune / Oppenheimer — large typography, controlled pacing, silence before impact
2.3 Design Personality
Dark. Gold. Precise. Fast.
Feels like a broadcast — not a portfolio template
Every animation has intent — nothing decorates, everything communicates

6. Animations — 21st.dev Reference
Source animation patterns from https://21st.dev/home and implement with GSAP + Framer Motion. Below are the animations to use per section.
6.1 Hero Section Animations
Name reveal
Staggered letter-by-letter reveal, Bebas Neue, sliding up from below (translateY 60px → 0, opacity 0 → 1, 0.8s ease, 0.05s stagger per char)
Tag line
Typewriter effect using GSAP TextPlugin — types out the role text character by character
Speed lines
SVG lines animate from right edge, width 0 → full, opacity 0 → 0.6, staggered 0.1s — like the car just blurred past
Scroll indicator
Pulsing chevron-down icon with infinite bounce animation (translateY 0 → 8px, 1s ease-in-out infinite)
Ticker
CSS marquee animation — tech stack scrolling left infinitely at 18s linear
6.2 3D Scene Animations
Car intro
Car fades in at z: -80, moves to z: -20 over 2s on page load. Subtle idle sway on Y axis (sin wave, amplitude 0.02, 2s period)
Scroll drive
GSAP ScrollTrigger maps scroll 0–100% to car Z position from -20 to +120. Car also tilts forward (rotationX) as speed increases
Floodlight flicker
SpotLight intensity lerps from 0.8 to 1.2 with random noise on mount — subtle, realistic
Camera drift
Camera X position drifts ±0.5 units sinusoidally as user scrolls — gives the broadcast camera feeling
Bloom pulse
Bloom effect strength increases on scroll from 0.3 to 0.8 — track lights glow harder as car accelerates
6.3 Content Overlay Animations
Section fade
Each section fades up: opacity 0 → 1, translateY 40px → 0, 0.9s ease. Triggered by IntersectionObserver at 20% threshold
Stats counter
Numbers count up from 0 to target value over 2s using GSAP countTo — triggered on scroll into view
Skill bars
Width animates from 0% to target% over 1.4s cubic-bezier(0.25, 1, 0.5, 1) when skills section enters viewport
Project cards
Cards stagger in: first card 0ms, second 150ms, third 300ms, fourth 450ms. Slight perspective tilt on hover (rotateX 3deg, rotateY -3deg)
Nav links
Underline slides in from left on hover — scaleX 0 → 1, transform-origin left
CTA button
Magnetic hover: button follows cursor within 40px radius, snaps back on mouse leave (GSAP quickTo)
6.4 Page Transitions
Use Framer Motion's AnimatePresence for route transitions
Exit: opacity 1 → 0, translateY 0 → -20px, 0.3s ease-in
Enter: opacity 0 → 1, translateY 20px → 0, 0.5s ease-out

9. 3D Scene Specification
9.1 Camera
Type
PerspectiveCamera, FOV 60
Initial position
x: 0, y: 2.5, z: 8 (behind and above the car)
Look at
x: 0, y: 1, z: 0 (car center mass)
Scroll behavior
z moves from 8 → -5 as user scrolls (camera chases car)
Drift
x oscillates ±0.3 sinusoidally over scroll — broadcast camera feel
9.2 Lighting Setup
AmbientLight
intensity: 0.05, color: #0a0a0a — near-total darkness
HemisphereLight
sky: #1a1a2e, ground: #0a0a0a, intensity: 0.1
SpotLight (key)
position above car, color: #FFD700, intensity: 2, angle: 0.3, penumbra: 0.5, castShadow: true
SpotLight (rim)
behind car, color: #4466FF, intensity: 1.5 — blue rim on rear wing
PointLight (front)
z: 10 ahead of car, color: #FF8800, intensity: 0.8 — headlamp warmth
Track floodlights
8x SpotLights on poles, color: #FFF5E0, intensity: 3, pointing down
9.3 Track — Procedural Construction
Road surface
PlaneGeometry 20 units wide × 200 units long. Material: MeshStandardMaterial, color #1a1a1a, roughness 0.9, metalness 0.1
Lane markings
BoxGeometry strips, color #FFFFFF, opacity 0.7, spaced every 8 units
Kerbs (left)
BoxGeometry, alternating red #CC0000 and white #FFFFFF segments, 0.5 wide × 0.1 tall
Kerbs (right)
Same as left
Armco barriers
BoxGeometry, color #C0C0C0, metalness 0.8, roughness 0.3, 0.4 wide × 1 tall
Floodlight poles
CylinderGeometry, color #888888, height 12 units, 8 poles each side
Fog
FogExp2, color #0a0a0a, density 0.015 — car disappears into darkness ahead
9.4 Post-Processing
Bloom
luminanceThreshold: 0.2, intensity: 0.8 on load → 1.4 at full scroll
Vignette
offset: 0.3, darkness: 0.8 — cinematic edge darkening
ChromaticAberration
offset: [0.001, 0.001] — subtle lens distortion
ToneMapping
ACESFilmicToneMapping — gives filmic color grading
9.5 Car Scroll Behavior
Scroll 0%
Car at z: 0, rotationX: 0 — idling on grid
Scroll 25%
Car at z: 30, subtle forward tilt rotationX: -0.02
Scroll 50%
Car at z: 60, rotationX: -0.04, camera drift begins
Scroll 75%
Car at z: 90, bloom intensity peaks
Scroll 100%
Car at z: 120, disappears into fog
